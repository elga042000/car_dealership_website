## Project Overview

This project is a modern car dealership website built to help users explore available cars, compare models, view pricing, and book test drives.

The website also includes an AI-style assistant panel that can respond to common user questions such as:

*Which is the cheapest model?
*Suggest a family car
*Which has the highest range?
*Show pricing
*Book a test drive
*Show features
*Open details of Elentra R9

## Tech Stack

### Frontend

* React
* Vite
* Tailwind CSS
* Framer Motion
* React Icons

### Backend

* Node.js
* Express.js
* MongoDB Atlas
* Mongoose

### Deployment

* Vercel for frontend
* Vercel serverless functions for backend

---

## Why I Chose This Stack

* React makes it easier to build reusable UI components.
* Vite provides faster local development and build times.
* Tailwind CSS allows quick styling without writing large CSS files.
* Framer Motion helps create smooth animations and a more premium feel.
* MongoDB is flexible for storing booking form data.
* Vercel is simple to deploy and works well with React projects.

## Setup Instructions

### 1. Clone the Repository

```bash
git clone <https://github.com/elga042000/car_dealership_website>
cd <client>
```

### 2. Install Frontend Dependencies

```bash
npm install
```

### 3. Install Backend Dependencies

If backend is inside a separate folder:

```bash
cd server
npm install
```

### 4. Create Environment Variables

Create a `.env` file in the backend folder:

```env
MONGODB_URI=your_mongodb_connection_string
PORT=5000
```

If using Vercel, also add the same environment variable in the Vercel dashboard.

### 5. Run the Frontend

```bash
npm run dev
```

### 6. Run the Backend

```bash
npm start
```
